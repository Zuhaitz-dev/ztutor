# مرحباً، Python

نفس الهدف كدرس C، لكن بلغة مختلفة.

اجعل البرنامج يطبع بالضبط:

```text
hello world
```

**مقارنةً بـ C:**

| C | Python |
|---|--------|
| `#include <stdio.h>` | *(غير مطلوب)* |
| `printf("...\n")` | `print("...")` |
| `return 0;` | *(غير مطلوب)* |

بايثون يتولى السطر الجديد والخروج تلقائياً. فقط النص داخل `print` يحتاج إلى تغيير.

## الجواب

```python
def main():
    print("hello world")

if __name__ == "__main__":
    main()
```
